========================================================================
    STATIC LIBRARY : xixEngine Project Overview
========================================================================

The xixEngine for 2D games as BlocksGame.

xixEngine.vcxproj
    This is the main project file for VC++ project.
    It contains information about the version of Visual C++, and
    information about the platforms, configurations, and project features.

xixEngine.vcxproj.filters
    This is the filters file for VC++ projects.
    It contains information about the association between the files in the project 
    and the filters. This association is used in the IDE to show grouping of files with
    similar extensions under a specific node (for e.g. ".cpp" files are associated with the
    "Source Files" filter).
